Welcome to textform!
====================

.. py:module:: textform

A data transformation pipeline module based on the seminal
`Potter's Wheel <http://control.cs.berkeley.edu/pwheel-vldb.pdf>`_ data wrangling formalism.
The name is a portmanteau of "text" and "transform".
